# Srihitha Portfolio

A minimalist dark-themed personal portfolio built with React.

## Getting Started

1. Install dependencies:
   npm install

2. Run locally:
   npm start

3. Build for production:
   npm run build

## Deploy to Netlify / Vercel
- Push to GitHub, then connect the repo to Netlify or Vercel.
- Build command: npm run build
- Publish directory: build

## Customise
- Replace the "SR" initials in App.jsx with your actual photo
- Update GitHub / LinkedIn / Resume href links in the Contact section
- Edit project titles, descriptions, and tags in the `projects` array
